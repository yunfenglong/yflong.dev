<h2>Deploy With GitHub Pages</h2>
<p>GitHub Pages is the simplest option for getting your first static site online.</p>
<h3>1. Create a repository</h3>
<ul><li>In GitHub, click <code>+</code> and choose <code>New repository</code>.</li><li>Use a name like <code>my-first-website</code>.</li><li>Set it to <code>Public</code> if you use the free GitHub Pages setup.</li><li>Create the repo.</li></ul>
<h3>2. Upload your site files</h3>
<ul><li>In the repo, click <code>Add file</code> -&gt; <code>Upload files</code>.</li><li>Upload <code>index.html</code> (or your project files).</li><li>Commit with a message like <code>feat: add initial website files</code>.</li></ul>
<h3>3. Enable Pages</h3>
<ul><li>Open <code>Settings</code> -&gt; <code>Pages</code>.</li><li>Under <code>Build and deployment</code>, choose <code>Deploy from a branch</code>.</li><li>Set branch to <code>main</code> and folder to <code>/(root)</code>.</li><li>Click <code>Save</code>.</li></ul>
<h3>4. Open your live URL</h3>
<p>GitHub will publish to:</p>
<p><code>https://&lt;your-github-username&gt;.github.io/&lt;your-repository-name&gt;/</code></p>
<p>If you see a 404 at first launch, wait a few minutes and refresh.</p>
<h2>Deploy With Cloudflare Pages</h2>
<p>Cloudflare Pages gives faster global delivery and a clean Git-based workflow.</p>
<h3>1. Create a Cloudflare account</h3>
<p>Sign up at <a href="https://dash.cloudflare.com/sign-up" target="_blank" rel="noopener noreferrer">dash.cloudflare.com/sign-up</a>.</p>
<h3>2. Connect Cloudflare Pages to GitHub</h3>
<ul><li>Open Cloudflare dashboard.</li><li>Go to <code>Workers &amp; Pages</code> -&gt; <code>Create application</code> -&gt; <code>Pages</code> -&gt; <code>Connect to Git</code>.</li><li>Authorize GitHub and select your repository.</li></ul>
<h3>3. Configure build settings</h3>
<p>For a plain HTML/CSS/JS website:</p>
<ul><li>Framework preset: <code>None</code></li><li>Build command: leave blank</li><li>Build output directory: leave blank or <code>/</code></li></ul>
<p>Then click <code>Save and Deploy</code>.</p>
<h3>4. Open your Cloudflare URL</h3>
<p>Your site will be available at:</p>
<p><code>https://&lt;project-name&gt;.pages.dev</code></p>
<h2>Automate Deployment With GitHub Actions</h2>
<p>Manual uploads are fine initially. For long-term maintainability, automate deployment with CI/CD.</p>
<p>Create <code>.github/workflows/deploy.yml</code>:</p>
<pre><code class="language-yaml">name: deploy

on:
  push:
    branches: [main]
  workflow_dispatch:

jobs:
  docs:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 22

      - name: Install Dependencies
        run: npm ci

      - name: Build VuePress site
        run: npm run docs:build

      - name: Deploy to GitHub Pages
        uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: gh-pages
          build_dir: docs/.vuepress/dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}</code></pre>
<p>Commit and push:</p>
<pre><code class="language-bash">git add .github/workflows/deploy.yml
git commit -m &quot;ci: add GitHub Actions workflow for deployment&quot;
git push</code></pre>
<p>Then set GitHub Pages source to the <code>gh-pages</code> branch.</p>
<h2>Q&amp;A and Troubleshooting</h2>
<h3>Why am I seeing 404 Not Found?</h3>
<ul><li>Wait 5 to 10 minutes after first deployment.</li><li>Ensure your entry file is <code>index.html</code>.</li><li>Confirm the deployment branch/folder is correct.</li><li>For free GitHub Pages, ensure repository visibility is <code>Public</code>.</li></ul>
<h3>Why are CSS or JavaScript files not loading?</h3>
<ul><li>Check path correctness in <code>href</code> and <code>src</code>.</li><li>Check filename case sensitivity.</li><li>If using a framework, inspect CI build logs.</li></ul>
<h3>How do I use a custom domain?</h3>
<ul><li>GitHub Pages: set custom domain under <code>Settings</code> -&gt; <code>Pages</code>, then add DNS records at your registrar.</li><li>Cloudflare Pages: use the <code>Custom domains</code> flow in your project.</li></ul>
<h3>Which should I choose?</h3>
